# 🛒 Анализатор прилавков

Система автоматической проверки правильности выкладки товаров на основе компьютерного зрения.

## Описание

Мерчендайзер фотографирует прилавок → система сравнивает фото с планограммой → формирует отчёт о нарушениях.

**Вдохновлено проектом:** [detecting-beer (YYalexx)](https://github.com/Yyalexx/detecting-beer) — система подсчёта товаров на витрине

**Ключевое отличие от оригинала:** вместо дообучения YOLOv5 (которое требует размеченного датасета) используется мультимодальная LLM (Claude Vision), что позволяет запустить MVP без единого обучающего примера.

## Архитектура

```
Фото прилавка
      │
      ▼
┌─────────────┐
│  Streamlit  │  ← веб-интерфейс
│   frontend  │
└──────┬──────┘
       │ изображение + планограмма
       ▼
┌─────────────────┐
│  Claude Vision  │  ← claude-sonnet-4-6
│  (Anthropic API)│     распознаёт товары,
└──────┬──────────┘     определяет порядок
       │ JSON с результатами
       ▼
┌─────────────┐
│   Отчёт     │  ← % соответствия, нарушения,
│  выкладки   │     детали по полкам
└─────────────┘
```

## Функциональность

- **Редактор планограммы** — задать эталонный порядок товаров по полкам (3 линейки, 5–8 товаров каждая), сохранить/загрузить JSON
- **Анализ фото** — загрузить фото прилавка, получить отчёт: что найдено, чего не хватает, нарушен ли порядок
- **История** — журнал всех проверок с возможностью экспорта отчётов

## Запуск

```bash
pip install -r requirements.txt
streamlit run app.py
```

При первом запуске введите Anthropic API ключ в настройках (раздел «Анализ фото»).

## Пример использования

1. Перейдите в **📋 Планограмма**
2. Задайте полки и товары (например: *Полка 1: Кола 0.5, Спрайт 0.5, Фанта 0.5*)
3. Перейдите в **📸 Анализ фото**, загрузите фото прилавка
4. Нажмите **Запустить анализ**
5. Получите детальный отчёт о соответствии

## Стек

- Python 3.10+
- Streamlit — веб-интерфейс
- Anthropic Claude Vision API — распознавание товаров
- Pillow — предобработка изображений

## Сравнение с оригинальным репозиторием

| | detecting-beer | shelf-analyzer |
|---|---|---|
| Модель | YOLOv5 (fine-tuned) | Claude Vision API |
| Датасет | Нужна разметка | Не нужен |
| Инфраструктура | RabbitMQ + Docker Compose | Streamlit (один файл) |
| Что определяет | Количество бутылок | Наличие + порядок товаров |
| Проверка планограммы | ❌ | ✅ |
| Время до запуска | Недели (разметка + обучение) | Минуты |
