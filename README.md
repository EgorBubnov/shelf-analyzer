## О проекте

**Shelf Analyzer** — это веб-приложение, предназначенное для автоматизированного анализа выкладки товаров на торговых полках.

В основе работы приложения лежит сравнение фотографии реальной полки с заранее подготовленной планограммой. Пользователь формирует эталонную схему расположения товаров, после чего загружает фотографию полки. Изображение анализируется мультимодальной моделью **Claude Vision**, которая определяет расположение товаров и возвращает результат в структурированном виде.

После обработки выполняется сравнение фактической выкладки с планограммой. На основе полученных данных формируется подробный отчет, позволяющий быстро оценить соответствие выкладки установленным требованиям.

Проект разрабатывался как исследование возможности применения современных мультимодальных языковых моделей для решения задач контроля выкладки товаров без обучения специализированных моделей компьютерного зрения.

---

## Основные возможности

* создание и редактирование планограмм;
* сохранение и загрузка планограмм в формате JSON;
* анализ фотографий торговых полок;
* автоматическое распознавание расположения товаров;
* сравнение результатов анализа с планограммой;
* определение отсутствующих товаров;
* выявление лишних товаров;
* проверка порядка расположения продукции;
* вычисление процента соответствия планограмме;
* формирование подробного отчета;
* сохранение истории выполненных анализов.

## Интерфейс приложения

Приложение состоит из нескольких разделов.

### Главная страница

Содержит краткое описание проекта и позволяет перейти к основным разделам приложения.

---

### Планограмма

В данном разделе пользователь может создать новую планограмму, определив список товаров для каждой полки и порядок их расположения.

Созданные планограммы сохраняются в формате JSON и могут использоваться повторно при последующих проверках.

---

### Анализ

После загрузки фотографии приложение обращается к модели Claude Vision для анализа изображения.

На основании полученного ответа выполняется сравнение фактической выкладки с планограммой. По результатам анализа формируется отчет, содержащий информацию обо всех обнаруженных несоответствиях.

---

### История

Все выполненные проверки сохраняются.

Это позволяет повторно просматривать результаты анализа без необходимости заново загружать изображения.

## Установка

Клонируйте репозиторий:

```bash
git clone https://github.com/EgorBubnov/shelf-analyzer.git
```

Перейдите в каталог проекта:

```bash
cd shelf-analyzer
```

Установите необходимые зависимости:

```bash
pip install -r requirements.txt
```

Запустите приложение:

```bash
streamlit run app.py
```

После запуска приложение будет доступно по адресу:

```
http://localhost:8501
```

## Использование

1. Создайте новую планограмму или загрузите существующую.
2. Укажите API-ключ Anthropic.
3. Загрузите фотографию торговой полки.
4. Запустите анализ.
5. Ознакомьтесь с отчетом.

По результатам обработки отображаются:

* общий процент соответствия планограмме;
* найденные товары;
* отсутствующие товары;
* лишние товары;
* нарушения порядка расположения продукции;
* итоговое заключение по результатам проверки.
